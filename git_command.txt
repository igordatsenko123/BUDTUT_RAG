Обновить ГІТ

# 1. Проверить текущий статус (изменённые файлы и т.д.)
git status

# 2. Добавить все изменения в индекс (можно указать конкретные файлы вместо .)
git add .

# 3. Зафиксировать изменения с комментарием
git commit -m "Твой комментарий про изменения"

# 4. Отправить изменения на GitHub (в ветку, например, main)
git push origin main
